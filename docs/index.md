# Bienvenue sur la documentation sur l'utilisation de l'IA

Bienvenue sur ce site de documentation dédié à l'utilisation de l'Intelligence Artificielle.

L'objectif de ce site est de rassembler des ressources, bonnes pratiques et guides pour vous aider à tirer le meilleur parti des outils d'IA disponibles aujourd'hui.

## Que trouverez-vous ici ?

- **Introduction** : Comprendre ce qu'est l'IA et comment fonctionnent les grands modèles de langage (LLM).
- **Bonnes pratiques** : Apprendre à rédiger de bons prompts et à évaluer les réponses générées.
- **Outils** : Un panorama des principaux outils d'IA disponibles.
- **Ressources** : Une sélection de liens, articles et formations pour aller plus loin.

## Pour commencer

Si vous débutez, nous vous recommandons de commencer par la section [Introduction](introduction/quest-ce-que-lia.md) pour acquérir les bases, puis de consulter les [Bonnes pratiques](bonnes-pratiques/rediger-des-prompts.md) pour utiliser l'IA efficacement.
