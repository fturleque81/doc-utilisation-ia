# Vérifier les réponses de l'IA

L'IA peut produire des réponses incorrectes, biaisées ou obsolètes. Apprendre à évaluer et vérifier les réponses est une compétence essentielle.

## Pourquoi vérifier les réponses ?

Les modèles de langage sont capables de générer des réponses convaincantes qui peuvent néanmoins être **factuellement incorrectes**. Ce phénomène est appelé **hallucination**.

Parmi les erreurs courantes :

- Faits inventés ou inexacts
- Citations attribuées à de mauvaises personnes
- Dates et chiffres erronés
- Références bibliographiques inexistantes
- Conseils juridiques ou médicaux incorrects

## Comment vérifier une réponse ?

### 1. Recouper avec des sources fiables

Pour toute information importante, vérifiez auprès de sources reconnues :
- Sites officiels (gouvernement, organisations internationales)
- Publications académiques (Google Scholar, PubMed…)
- Encyclopédies de référence (Wikipédia avec précaution, Larousse…)

### 2. Demander des sources à l'IA

Vous pouvez demander au modèle de justifier ses affirmations :

```
Quelles sont tes sources pour cette affirmation ?
Peux-tu me donner les références précises ?
```

!!! warning "Attention"
    Le modèle peut inventer des références bibliographiques qui n'existent pas. Vérifiez toujours l'existence réelle des sources citées.

### 3. Croiser plusieurs outils

Comparez les réponses de différents modèles ou outils pour identifier les divergences.

### 4. Faire preuve d'esprit critique

Appliquez les mêmes règles qu'avec n'importe quelle source :
- Qui a produit cette information ?
- La réponse est-elle cohérente avec ce que je sais ?
- Y a-t-il des éléments suspects ou trop simplistes ?

## Ce que l'IA ne peut pas faire

| Limitation | Explication |
|-----------|-------------|
| Accéder à Internet (sauf outils dédiés) | Les modèles de base travaillent avec leurs données d'entraînement |
| Connaître l'actualité récente | Les données s'arrêtent à une date de coupure |
| Remplacer un expert | Pour des sujets sensibles (médecine, droit), consultez un professionnel |
| Être objectif | Les modèles peuvent reproduire des biais |

!!! tip "Règle d'or"
    Utilisez l'IA comme un point de départ, pas comme une source définitive. Vérifiez toujours les informations critiques.
