# Panorama des outils d'IA

Il existe aujourd'hui de nombreux outils d'IA accessibles au grand public. Cette page présente un aperçu des principaux outils par catégorie.

## Assistants conversationnels (chatbots)

Ces outils permettent de dialoguer avec un modèle de langage pour obtenir de l'aide sur de nombreuses tâches.

| Outil | Éditeur | Lien |
|-------|---------|------|
| ChatGPT | OpenAI | [chat.openai.com](https://chat.openai.com) |
| Gemini | Google | [gemini.google.com](https://gemini.google.com) |
| Claude | Anthropic | [claude.ai](https://claude.ai) |
| Copilot | Microsoft | [copilot.microsoft.com](https://copilot.microsoft.com) |

## Critères de choix d'un outil

Lors du choix d'un outil d'IA, pensez à évaluer :

- **Confidentialité des données** : Vos données sont-elles utilisées pour entraîner le modèle ?
- **Coût** : L'outil est-il gratuit, freemium ou payant ?
- **Qualité des réponses** : Le modèle est-il adapté à votre cas d'usage ?
- **Langue** : Le modèle gère-t-il bien le français ?
- **Intégrations** : S'intègre-t-il à vos outils existants ?

!!! note "Évolution rapide"
    Le paysage des outils d'IA évolue très rapidement. De nouveaux outils apparaissent régulièrement. Cette liste est donnée à titre indicatif.
